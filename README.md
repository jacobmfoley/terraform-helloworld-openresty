# Terraform Helloworld Openresty

Simple hello world deployed to AWS using Nginx/Openresty.

### Prerequisites

- Terraform

```
# OSX Homebrew
brew install terraform
```

### Installing



```
cd $projectDir/providers/aws/us-west-2/vpc_web0/
terraform init
terraform plan
terraform apply
```
